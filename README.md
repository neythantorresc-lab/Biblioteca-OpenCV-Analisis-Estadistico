# Biblioteca-OpenCV-Analisis-Estadistico
Aplicación en Python usando OpenCV para análisis estadístico en tiempo real.
Analizador Estadístico en Python
Descripción

Este proyecto es un analizador estadístico desarrollado en Python que permite calcular medidas estadísticas básicas y generar gráficos a partir de un conjunto de datos.

Tecnologías utilizadas

-Python 3.x

-NumPy

-Pandas

-Matplotlib

Cómo ejecutar el programa.

1. Clonar el repositorio:
"git clone https://github.com/tu-usuario/tu-repositorio.git"

2. Instalar dependencias:
"pip install numpy pandas matplotlib"

3. Ejecutar:
"python analizador_estadistico.py"

Funcionalidades:

-Cálculo de media

-Cálculo de mediana

-Cálculo de moda

-Generación de gráficos estadísticos
